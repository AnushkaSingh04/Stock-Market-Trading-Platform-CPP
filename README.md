# 📈 Stock Market Trading Simulator (C++)

A fully interactive, console-based **Stock Market Trading Simulator** built using C++.  
This project simulates real-time stock trading with portfolio management, live market updates, trade history tracking, and animated terminal UI elements.

## 🏗 Project Architecture

The system is designed using modular object-oriented architecture:

- **Stock** → Handles stock details and price fluctuation logic  
- **Portfolio** → Manages user balance, holdings, and trade history  
- **Market** → Maintains stock listings and real-time updates  
- **Trade** → Stores transaction details with timestamps  

This separation ensures clean code structure and scalability.
